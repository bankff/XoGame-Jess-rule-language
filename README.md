## INSTALL Jess

Download https://jess.sandia.gov/jess/index.shtml

## Running the app

1.Move file XoGameProjectAI.clp to `Jess71p2\examples\jess`

2.Run on cmd
```
bin\jess.bat examples\jess\XoGameProjectAI.clp
```
3.Enjoy
